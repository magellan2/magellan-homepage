Source Code

The Magellan source code is published under the terms of the GNU General Public License (GPL). It is available via public CVS access.

CVS Archive

To access the Magellan CVS archive use this CVSROOT variable: CVSROOT=:pserver:readonly@eressea.upb.de:/home/cvs/magellan. The required password is readonly.

The available CVS modules are: src, res, www, help, doc, themes, and patcher.

The src module can also be browsed online.
