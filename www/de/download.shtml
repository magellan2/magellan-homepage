Das Programm

Die neuesten Versionen von Magellan befinden sich jetzt auf SourceForge

Den Download magellan.jar sollte man als Datei auf dem eigenen Rechner abspeichern, diese Datei muss nicht dekomprimiert werden oder ähnliches (siehe auch F.A.Q.)

Magellan v2

Magellan2 ist eine konsequente Weiterentwicklung des Magellan Clients. Die Änderungen an diesem Client sind so umfassend, dass wir uns entschlossen haben, eine neue Versionsnummer zu vergeben. Du kannst dieses Programm von der aktuellen Entwicklungssite herunterladen. Es befindet sich momentan noch im Beta-Stadium, besitzt aber alle Funktionen des alten Clients und einige neue Features.

Java

Um Magellan starten zu können, muss erst mal Java - das sogenannte Java Runtime Environment (JRE) - installiert werden. Man kann auch JREs von anderen Herstellern als SUN verwenden, es muss aber mindestens Version 1.4 sein - unter Linux hat sich übrigens anscheinend IBM's JRE besser bewährt als das von SUN.

Entwicklerversion - Beta Test

Beginnend Mitte 2006 hat sich das Entwicklerteam veraendert. Fuer die Uebergangszeit (bis wir uns richtig in die Programmierung von Magellan eingearbeitet haben) werden Aenderungen an Magellan nicht sofort auf SourceForge verfuegbar gemacht. Die letzte halbwegs getestete Version findet sich auf der Seite von Fiete. Tester sind hochwillkommen.

Die neueste Entwicklungsversion von Magellan v2 gibt es auf Thoralfs Server.
