Application

The newest versions of Magellan nowadays can be found at eressea.sourceforge.net

Simply save the magellan.jar file, it does not need to be de-compressed (also see the F.A.Q.)

This is the latest version, containing the latest bug-fixes and bugs. It is rebuilt instantly every time the source code of Magellan is modified.

Alternatively, you can download the Eressea Installer containing Magellan, Java, the patcher, the Eressea and Magellan documentation, and a sample report. It does not always include the latest version of Magellan but updating Magellan is easy using the patcher.

Please use the patcher to keep your copy of Magellan always up-to-date.

Java

To run Magellan you need to install Java - the Java Runtime Environment (JRE) to be precise. If you want to use a JRE from a different vendor than Sun, make sure you have version 1.3 or newer. By the way, we learned that under Linux IBM's JRE performs better than the one from Sun.

Documentation

The help documents all Magellan features and is accessible via the Help entry in the Extras menu.

Installation:

  • Download the JavaHelp Plug-In; it contains the necessary components to display the help.
    (To get the latest version of JavaHelp, download the JavaHelp package and take the file jh.jar from the javahelp\lib subdirectory)
  • Include the JavaHelp Plug-In (i.e. the file jh.jar) in a resource path.
  • Get the actual Magellan documentation.
  • Include the help (i.e. the file magellan-help.jar) in a resource path.
  • Restart Magellan.
  • Now you can access the help via the Help entry in the Extras menu.

Please use the patcher to keep your copy of the help always up-to-date.

Patcher

The Magellan Patcher allows to update your version of Magellan, the help or the patcher itself without downloading the whole jar packages.
Make sure that the patched package is neither being executed nor accessed. E.g. you lose your program settings when patching a running Magellan.

Graphic Sets

With graphic sets you can modify the appearance of the map displayed by Magellan. You can use your favorite map graphics in case you do not like the default ones. Currently, there are two graphic sets available:

  • E-Map graphic set by Henning Zahn. This is how Magellan looked some time ago. Last modification: 2001.01.06
  • Nugman graphic set by Roland Behme. Contains roads. Last modification: 2001.02.08
  • Ageless graphic set by Jake Hofer. Contains roads. Last modification: 2002.04.18

After downloading, include the graphic set in a resource path and restart Magellan.

If you want to create your own graphic set, please refer to the documentation.

Look&Feel and Skins

You can change Magellan's appearance, the so-called Look&Feel, on the System panel in the Magellan options. Java's three default Look&Feels are: Metal, CDE/Motif, and Windows. With the Skin Look&Feel Plug-In many additional Look&Feels become available giving Magellan the skin of your choice (e.g. a Windows XP look).

Installation:

  • Download the file skinlf.jar.
    (To get the latest version which may be required for some skins, download the Skin Look and Feel package from L2F Productions and extract the file skinlf.jar from it)
  • Place the file skinlf.jar into your Magellan directory (e.g. C:\magellan\).
  • Get your favourite theme pack from javootoo.com.
  • Place the theme packs into the skins subdirectory of your Magellan directory (e.g. C:\magellan\skins\).
  • Restart Magellan.
  • Choose your theme pack from the Look&Feel list in the System tab in the Magellan options.

Skill Charts Plug-In

This add-on displays all skills of your faction's units as a concise diagram giving you both a quick overview and many details about all the skills and knowledge in your faction.

Installation: Download the file chart.jar and include it into a resource path. Restart Magellan.
The sourcecode for this package is available here.

Support for other PBeMs

To use Magellan for other PBeMs than Eressea, Magellan needs to operate on different rules and orders. These modifications are available as extensions:

  • Verdanon extension, maintained by Jens Schulze.

Installation:

  • Download the extension.
  • Include it in a resource path.
  • Restart Magellan.
