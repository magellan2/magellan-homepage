The Program

Newest Version of Magellan can be found at SourceForge

The Download magellan.jar should be saved as a file, you do not need to unzip it or use other decompressing-tools (see F.A.Q.)

The complete changelog regarding the last changes on Magellan is available at the website of Fiete.

Magellan v2

Magellan2 is a consistent further development of the original Magellan client. The changes are so extensive, that we've decided to give it a new version number. You can download the current version of it from the development site. It is currently a beta version but contains all functions of Magellan v1 and some more new features.

Java

To run Magellan you have to install the Java Runtime Environment (JRE) - first. JREs from other providers than Sun are OK and working, but it should be version 1.4 or newer - by the way: under Linux IBM's JRE seems to be better than SUN's.
