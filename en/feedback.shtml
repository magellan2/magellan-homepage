Bugs in Magellan

A good starting point in searching for bugs are the error messages produced by Magellan. They are in the file errors.txt in the Magellan directory. Plz provide us with this file. We allways try to reproduce a problem and this file can help a lot. Sometimes we hardly need a CR to reproduce a problem - we will keep them save and warm and will delete them after solving the problem.

Bugs and Feature-requests = Development

We try to use the wishes of Magelan-users to keep a good direction in the Magellan development process. For this reason we installed a bugtracker named Mantis. That's the right place for bugs and feature request - allthough it´s in german, we will be able to read your mind ;-)
